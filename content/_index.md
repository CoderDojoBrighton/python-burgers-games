# Pygame Zero: Burgers Game

<img src="/python-burgers-games/_images/burgers_complete.png" alt="" style="width:600px;"/>

In this tutorial we will make a game about catching falling ingredients and building burgers.

If you are just starting out with Python, start with the Flappy Bird tutorial first.

This tutorial assumes you have already completed the Flappy Bird tutorial, so we will skip some of the really basic stuff.

## Getting Started

Check the beginning of the Flappy Bird tutorial for instructions on installing Mu.

## Important tip for using Mu

The first time you save your work in Mu it will open the save dialog in a folder called `mu_code`. It is important you save in that exact folder. If you do not, Mu will not be able to run your games.

## Installing Burger Images

- Download the tutorial assets by clicking this [link](https://kinolien.github.io/gitzip/?download=https://github.com/andyborrell/pygamezero_burgers/tree/master/images). You can close that page once the download finishes.
- Click on the **Images** button in the Mu editor. This will open the directory where Pygame Zero looks for images.
- Copy the `burgers` folder from the `images.zip` file you just downloaded into this directory.

Once those assets are in place, use the navigation to jump into Part 1 and build the game step by step.
